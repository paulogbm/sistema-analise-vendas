# 📊 SalesVision BI

Sistema de Análise Inteligente de Dados de Vendas desenvolvido para a disciplina de Teste de Software.

## Descrição

O SalesVision BI é uma plataforma web para análise de dados de vendas a partir de arquivos CSV. O sistema permite o envio de arquivos, processamento automático dos dados e exibição de métricas relevantes para auxiliar na tomada de decisões.

## Funcionalidades

* Login de acesso ao sistema
* Dashboard interativo
* Upload de arquivos CSV
* Análise automática de vendas
* Cálculo da receita total
* Identificação do produto líder em vendas
* Identificação do produto com menor desempenho
* Detecção de outliers (valores fora do padrão)
